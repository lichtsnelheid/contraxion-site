# Contraxion website

Statische GitHub Pages website voor Contraxion.

## Bestanden
- `index.html` - website structuur en content
- `styles.css` - kleuren, layout en responsive design
- `script.js` - mobiel menu en subtiele scroll-animaties
- `CNAME` - custom domein voor GitHub Pages

## Publiceren via GitHub Pages
1. Upload alle bestanden naar de repository.
2. Ga naar **Settings > Pages**.
3. Kies **Deploy from a branch**.
4. Branch: `main`; folder: `/root`.
5. Klik op **Save**.
